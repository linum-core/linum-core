## 🎯 OVERVIEW DO PROJETO

### O que é Linum Core?
Portfolio website profissional para **Linum Core** - empresa de consultoria e desenvolvimento de sistemas.

**Significado da marca:**
- "Linho" é uma planta usada para criar fios e tecidos
- Representa **conexão e união**
- **Lema:** "Existe uma conexão não percebida... Nós construímos ela, não criando sistemas mas restauramos conexões."

### Objetivo Principal
Criar um website que transmita:
- ✅ Seriedade e profissionalismo
- ✅ Capacidade real de desenvolver sistemas e consultoria
- ✅ Confiabilidade total
- ✅ Design diferente, interativo, bonito, moderno e elegante
- ✅ Atração para fechar contratos

---

## 🎨 IDENTIDADE VISUAL

### Filosofia de Design
- **Tema:** Sofisticação + Tecnologia + Simplicidade
- **Estilo:** Animações mostram vida, movimento e confiabilidade
- **Abordagem:** Design-first, respeitando padrões modernos

### Sentimento Desejado
Visitantes devem sentir impulso para:
1. Conhecer mais o trabalho realizado
2. Confiar na empresa
3. Fechar contrato com a Linum Core

## 🔤 TIPOGRAFIA

### Fonts Principais

| Uso | Font | Fallback | Weights |
|-----|------|----------|---------|
| **Headlines/Títulos** | Cinzel | serif | 400, 600, 700 |
| **Subtítulos** | Jomolhari | serif | 400 |
| **Body Text/Corpo** | Inter | sans-serif | 400, 500, 600 |
| **Código** | Fira Code | monospace | 400, 500 |

### Hierarquia Tipográfica
- **H1 (Cinzel 700):** Títulos principais da página
- **H2 (Cinzel 600):** Subtítulos de seção
- **H3 (Cinzel 400):** Títulos secundários
- **Body (Inter 400-600):** Texto principal
- **Small (Inter 400):** Rodapé, metadados

### Acessibilidade
- Mínimo 16px para body text em mobile
- Contrast ratio mínimo 4.5:1 para WCAG AA
- Line-height mínimo 1.5

## 🎬 ANIMAÇÕES & EFEITOS VISUAIS

### Efeitos Recomendados
- **Fade In/Out:** Transições suaves entre seções
- **Scroll Animations:** Elementos que aparecem ao rolar
- **Hover Effects:** Botões e cards com efeito de hover
- **Parallax:** Efeito de profundidade em hero section
- **3D Elements:** Inspiração em Resend (cubo 3D)
- **Loading Animations:** Transições suaves entre páginas

### Princípios
- Performance-first (usar GPU acceleration)
- Smooth (60fps)
- Acessível (respeitar `prefers-reduced-motion`)
- Sofisticado (não excessivo)
