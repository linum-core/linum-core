## 📋 Resumo Executivo

**Objetivo:** Criar um portfólio profissional e sofisticado para "Linum Core" que demonstre expertise em desenvolvimento de sistemas, manutenção, consultoria e prospecte novos clientes.

**CEO:** Gabriel Gomes

**Lema:** "Existe uma conexão não percebida… Nós construímos ela, não criando sistemas mas restauramos conexões."

---

## 🎯 Objetivos do Site

### Sentimento Desejado
Visitantes devem sentir:
- ✅ Que o proprietário e empresa são sérios e respeitáveis
- ✅ Design diferente, interativo, bonito, moderno e elegante
- ✅ Capacidade real de desenvolver sistemas e prestar consultoria
- ✅ Confiabilidade total
- ✅ Impulso para conhecer mais o trabalho
- ✅ Atração para fechar contrato com a empresa

---

## 🛠️ Requisitos Técnicos

### Stack Tecnológico
- **Framework:** Next.js 15
- **Styling:** Tailwind CSS 4
- **Language:** TypeScript 5
- **Características:**
  - ✅ Sempre preocupado com segurança
  - ✅ Ótimo ranqueamento SEO
  - ✅ Acessibilidade WCAG
  - ✅ Camada de integração com API
  - ✅ Responsivo para todos dispositivos
  - ✅ Baixa latência
  - ✅ Componentização e micro components
  - ✅ Otimização de performance

## 📄 Estrutura de Páginas

### Header (Todos as páginas)
- Logo
- Menu: Home, Portfólio, About, CEO, Competências, Contato, Feedbacks

### 1️⃣ Home (Hero)
- Frase: "Existe uma conexão não percebida…"
- Logo + Nome da Empresa
- CEO: Gabriel Gomes
- Animação prendente

### 2️⃣ Portfólio
- Cards de projetos
- Preview Hero/Screenshot
- Referência GitHub
- Link direto
- Status: Completo / In Progress
- Filtro por status

### 3️⃣ About (CEO)
- Foto do CEO
- Bio
- Skills em Gráfico de Radar
- Skills Técnicas, Códigos, Ferramentas, Relacionais

### 4️⃣ Competências
- Explicar o que a empresa pode realizar
- Seções descritivas

### 5️⃣ Feedbacks
- Cards em Carrossel/Loop
- Depoimentos de clientes

### 6️⃣ Contato
- Email, WhatsApp, Instagram, LinkedIn, GitHub

### Footer
- Links repetindo header
- Copywrite
- Links de contato
- Endereço
